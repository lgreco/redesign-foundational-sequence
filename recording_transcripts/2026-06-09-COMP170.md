## The Airplane Seating Problem

We started with a small plane: 9 seats in 3 rows, 3 seats per row, labeled A, B, C. The goal was to take a numbered list of passengers (0 through 11 for a 12-seat plane) and assign each one a seat label like "2B" or "4A." Before writing any code, we looked for the pattern.

## Spotting Two Patterns

With Bella's observation that the seat letters repeat in groups of three, and Erica's that the pattern cycles A, B, C, A, B, C …, we named two distinct behaviors:

- **Seat letter (the repeating pattern):** A, B, C, then back to A. The number of seats per row controls how often it repeats.
- **Row number (the grouping pattern):** stays the same for 3 passengers, then steps up by 1.

## Turning Patterns into Code

We translated both patterns into arithmetic:
- `passenger % seats_per_row` produces 0, 1, 2, 0, 1, 2 … — the seat position as a number. Adding that to 65 (the numeric code for the letter 'A') and converting with `chr()` gives the seat letter.
- `passenger // seats_per_row + 1` gives the row number. The `+ 1` shifts from starting at 0 to starting at 1, the way row numbers work on a real plane.

We wrote and ran the code, then used Python's f-string formatting to keep single- and double-digit passenger numbers neatly aligned. Changing `seats_per_row` from 3 to 4 instantly produced a correct layout for a wider plane — no other changes needed.
