## Tools

Students are encouraged to move away from IDLE toward a more capable environment. Two options were demonstrated side by side:

- **GitHub Codespaces**: a browser-based instance of VS Code that provisions a full Ubuntu virtual machine. No local install needed. Free tier provides well over enough compute hours for the course.
- **Local VS Code**: faster to launch and easier to customize; functionally identical to Codespaces.

Either option is acceptable. If students already use PyCharm, NetBeans, or another IDE they are comfortable with, they may continue using it.

The **Black** code formatter was installed and demonstrated. Right-clicking → "Format Document" automatically applies consistent spacing and line-breaking conventions. Major companies maintain detailed style guides so that all developers can read each other's code; the course will use a simpler "Programmer's Pact" shared as a reading assignment.

## Preview of Next Week

The central theme of the course is the **array** (Python: list) — the foundational data structure. The key insight: computers store data linearly, but humans think in two and three dimensions. Data structures are the interfaces that reconcile the computer's linear memory with our richer spatial models (org charts, airplane seating, stadium tiers). Next week will begin exploring that bridge through code.
