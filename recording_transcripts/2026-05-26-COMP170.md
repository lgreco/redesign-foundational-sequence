- `ls -al` — list directory contents with details

## Python Basics: Numbers vs. Strings

Inside the Python interactive shell (`python3`):

- `2 + 2` → `4` (integer addition)
- `"2" + "2"` → `"22"` (string concatenation — the `+` operator joins strings, not adds them)

Python accepts both single and double quotes for strings; they are equivalent.

## Compound Interest

Built up a compound interest computation step by step:

1. Year 1: `1000 * (1 + 0.045)` → $1,045
2. Year 2: `(1000 * (1.045)) * (1.045)`
3. Generalized: `1000 * (1.045) ** n` — the `**` operator is Python's exponentiation

## Separation of Concerns

Even in a small program, three concerns should eventually be kept separate:

- **Input** — gathering data from the user
- **Logic** — computing the result
- **Output** — presenting the result

Mixing all three in one block works for beginners but becomes a problem as programs grow. This separation will be a recurring theme in the course.
